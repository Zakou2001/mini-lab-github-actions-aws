# GitHub Actions – Pipelines CI/CD

## Objectif
Ce document décrit les bonnes pratiques et les étapes pour créer et maintenir des pipelines CI/CD avec **GitHub Actions** pour nos projets.

---

## Prérequis
- Un repository GitHub
- Droits d’écriture sur le repo
- Connaissances de base en Git
- Un runner GitHub (hébergé ou self-hosted)
